Privacy Policy

Last updated: July 2026

This application ("DearDariaCo Pinterest Poster") is a personal-use tool built to automatically publish pins to a single Pinterest account owned by the app's creator (DearDariaCo).

What this app does

This app connects to one Pinterest account via the Pinterest API to create pins (images, titles, descriptions, and links) on boards belonging to that same account. It does not interact with any other user's Pinterest account.

What data is accessed

The app accesses only the Pinterest account owner's own boards and pins, using an access token authorized directly by the account owner through Pinterest's official OAuth process. No data belonging to other Pinterest users is accessed, collected, or stored.

What data is stored

The app stores a simple log of which images and captions have already been posted, so it can avoid duplicate posts. This log contains no personal data about any third party.

Data sharing

No data collected or generated by this app is sold, shared, or transferred to any third party.

Contact

For questions about this policy, contact DearDariaCo through the shop's Etsy page.
